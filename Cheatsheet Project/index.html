Display and Position in CSS

Defining Block and Inline

Block: Elements that start a new line in the browser. Such as: <h1> <p> <ul>

Inline: Elements that continue the same line as their adjacent elements. Such as: <a> <b> <img>

Display

Allows to turn an inline element to a block element and viceversa

Inline: Block-level element acts like an inline

Block: Inline element acts like an block-level

Inline-block: Block-level element flows like an inline. Retains other features of a block-level element.

Position

Normal Flow:
(Default) Block- level elements appear in a new line. Each item appears lower down than the previous one.

Relative:
Moves an element from the static position it would be in normal flow. Doesn't affect sorrounding elements.

Absolute
Positions the element in relation to the containing element. Out of the normal flow. Moves with page scroll.

Fixed
Positions the element in relation to the browser window. Does not move with page scroll.

Z-Index: used to overlap elements. Value is a number. The higher the number, the closest is the element to the front

Floating
Takes the element out of the normal flow and positions left or right to a container. Other content flows around it. Block-level element.

Clear: allows that no element should touch the left o right hand sides of a box. Values: left, right, both, none.
